# Honari

Honari is an Android reading companion built with Jetpack Compose, Hilt, Room, Firebase, CameraX, and the Google Books API.

## Local setup

Create or update `local.properties` with your Google Books API key before running the app:

```properties
GOOGLE_BOOKS_API_KEY=your_api_key_here
```

`default_web_client_id` is defined in `app/src/main/res/values/strings.xml` with a placeholder for local compilation. Replace it with the value generated by your Firebase / Google Services setup when needed.
